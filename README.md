# Hello Cucumber

A hello world project for first contacts with the cucumber test library.

## Installation

Use the package manager [pnpm](https://pnpm.io) to install Hello Cucumber.

```bash
pnpm install
```

## Run Cucumber Tests

In your terminal, run the following command:


```python
pnpm run test
```
